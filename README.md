# Creative Newsletter Pro WordPress Theme

Complete WordPress theme for newsletter creators and digital product sellers.

## Features
- Modern purple/blue gradient design
- Fully responsive
- Custom post types (Newsletters, Products)
- Newsletter signup integration
- About page featuring Ranvijay Singh

## Installation
1. Download theme files
2. Upload to /wp-content/themes/
3. Activate in WordPress admin
4. Customize via WordPress Customizer
